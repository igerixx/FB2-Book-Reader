# Fb2 Book Reader

# Description
BookReader is an application that parses FB2 and other XML-like files, extracting and separating content such as paragraphs, sections, titles, images, and other structural elements. It is built on a custom optimized streaming parser designed for fast and efficient processing of large files.
The application includes user authentication with login and registration, maintains a history of books, and provides a clean, user-friendly interface with a dark theme for comfortable reading.
